Cách đúng là:
git add .
(chú ý có khoảng trắng giữa add và .)

Hoặc nếu chỉ muốn thêm 1 file cụ thể:

git add index.html


👉 Không cần thêm "Add" ở cuối lệnh git add.
Sau khi add xong, bạn có thể commit:

git commit -m "Thêm file index.html"


Và đẩy lên GitHub (nếu bạn đã kết nối remote):

git push origin main


⚠️ Nếu chưa kết nối repo GitHub thì bạn cần chạy thêm:

git remote add origin https://github.com/DuyNguyen1912/Code_du_an_NCKH_2024.git
git branch -M main
git push -u origin main
